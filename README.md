# dotfiles

A collection of config files

## Installs
- nvm
- sdkman
- nvim
- to.sh
- llvm
- jq

## Setup required
- configuring global gitignore
- to.sh aliases

## Installation

### Setup
Create a set of symbolic links e.g. `ln -s $(pwd)/.zshrc ~/.zshrc`
> run `make`

### Remove
> run `make clean`
